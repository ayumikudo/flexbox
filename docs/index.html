flex box Layout

基本のコード

＜span＞でマークアップした3つのフレックスアイテムを用意し、フォントサイズと背景色を変えて表示しています。各フレックスアイテムの横幅は指定していません。

  • 初期状態

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      <h4>
                  <span>★</span>
                  <span>もぎたてトマトの<br>美味しい食べ方</span>
                  <span>★</span>
                  </h4>
                
                      h4 {
                    
    display: flex;
    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                    color: #fff;
                    background-color: #e0de0a;
                    font-size: 4em;
                  }
                  h4 span:nth-child(2) {
                    background-color: #eaad38;
                    font-size: 3em;
                  }
                  h4 span:nth-child(3) {
                    color: #fff;
                    background-color: #a9de4c;
                    font-size: 2em;
                  }

横方向の位置揃えを指定する

フレックスアイテムの横方向の位置揃えは、フレックスコンテナ＜ｈ4＞のjustify-contentで次のように指定することができます。

  • 左揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    justify-content: flex-start;
    border: solid 10px #85cfee;
                  }

  • 中央揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    justify-content: center;
    border: solid 10px #85cfee;
                  }

  • 右揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    justify-content: flex-end;
    border: solid 10px #85cfee;
                  }

  • 均等割りでフレックスアイテムの間に余白を挿入

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    justify-content: space-between;
    border: solid 10px #85cfee;
                  }

  • 均等割りでフレックスアイテムの間と両サイドに余白を挿入
    ※ただし両サイドの余白は半分のサイズになります。

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    justify-content: space-around;
    border: solid 10px #85cfee;
                  }

余白の配置を調整する

フレックスコンテナ内の余白は、marginプロパティの「auto」を利用してどこに配置するかをコントロールすることができます。

  • 1つめの要素の左側に配置

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                  	
    margin-left: auto;
    }

  • 1つめの要素の右側に配置

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                  	
    margin-right: auto;
    }

  • 1つめと3つめの要素の左右に配置
    ※以下のように4つに分割されます。

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                    
    margin-left: auto;
    margin-right: auto;
    }
                    h4 span:nth-child(3) {
                      
    margin-left: auto;
    margin-right: auto;
    }

縦方向の位置揃えを指定する

フレックスアイテムの縦方向の位置揃えは、フレックスコンテナ＜ｈ4＞のalign-itemsで次のように指定することができます。

  • 上揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    align-items: flex-start;
    border: solid 10px #85cfee;
                  }

  • 中央揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    align-items: center;
    border: solid 10px #85cfee;
                  }

  • 下揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    align-items: flex-end;
    border: solid 10px #85cfee;
                  }

  • ベースライン揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    align-items: baseline;
    border: solid 10px #85cfee;
                  }

  • ストレッチ

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
    align-items: stretch;
    border: solid 10px #85cfee;
                  }

  • 1つめの要素のみ上揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                  	
    align-self: flex-start;
    }

  • 1つめの要素のみ中央揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                    
    align-self: center;
    }

  • 1つめの要素のみ下揃え

    ★ もぎたてトマトの
    美味しい食べ方     ★ 

                      h4 {
                    display: flex;
                    border: solid 10px #85cfee;
                  }
                  h4 span:nth-child(1) {
                    
    align-self: flex-end;
    }
